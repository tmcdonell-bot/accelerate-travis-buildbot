class Intrinsic arch where Source

During code generation we need to know the name of functions implementing certain intrinsic maths operations. Depending on the backend, these functions may not be implemented using the standard C math library.

This class allows a backend to provide a mapping from the C math library function name to the name of the function which should be called instead. The default implementation maps to the llvm intrinsic. For example:

sqrtf -> llvm.sqrt.f32 sqrt -> llvm.sqrt.f64
