data Nursery a Source

The nursery is primarily designed as a place to store device memory arrays that are no longer needed. Often it is quicker to reuse an existing array, rather than calling into the device API in order to allocate a fresh array.

The nursery is wrapped in an MVar so that several threads may safely access it concurrently.
