class Skeleton arch where Source

A class covering code generation for all of the primitive array operations. Client backends implement an instance of this class.

Minimal complete definition: * generate * fold, fold1, foldSeg, fold1Seg * scanl, scanl', scanl1, scanr, scanr', scanr1 * permute
