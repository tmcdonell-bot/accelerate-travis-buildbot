    class C a => C a where Source #

    This class lets us deal with units in a ring. isUnit tells us whether an element is a unit, while the other operations let us canonically write an element as a unit times another element.

    This instance shadows C with an instance of isUnit compatible with Accelerate.

    Minimal complete definition

    isUnit, (stdUnit | stdUnitInv)
