    extendedEuclid :: forall a. (C (Exp a), C (Exp a), C (Exp a), C (Exp a), Elt a) => (Exp a -> Exp a -> Exp a) -> Exp a -> Exp a -> Exp (a, (a, a)) Source #

    Implementation of the extended Euclid algorithm which computes, besides the greatest common divisor of integers a and b, the coefficients of Bézout's identity x and y such that:

    ax + by = gcd(a, b)
