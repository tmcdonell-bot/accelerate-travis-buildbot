fromIArray :: (IxShapeRepr (EltRepr ix) ~ EltRepr sh, IArray a e, Ix ix, Shape sh, Elt ix, Elt e) => a ix e -> Array sh e Source #

O(n). Convert an IArray to an Accelerate Array.

The index type ix of the IArray corresponds to the shape sh of the Accelerate Array in the following way:

DIM0 ~ ()
DIM1 ~ Int
DIM2 ~ (Int,Int)
DIM3 ~ (Int,Int,Int)

...and so forth.
