data EigType Source #

This type indicates which type of eigenvalue solver is used. It corresponds to the parameters used by legacy LAPACK implementations:

  • EigType1: \( A*x = lambda*B*x \)
  • EigType2: \( A*B*x = lambda*x \)
  • EigType3: \( B*A*x = lambda*x \)
